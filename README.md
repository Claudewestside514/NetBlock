## ❓ What is NetBlock?

NetBlock is a simple tool for Windows that lets you block internet access for specific apps. If you want certain programs to run without connecting to the web, NetBlock can help. This is useful for saving data, improving privacy, or stopping apps from updating automatically.

NetBlock works by controlling which applications can send or receive data over the internet. You pick the apps, and NetBlock does the rest in the background. It runs quietly on your PC without slowing it down.

## 🖥️ System Requirements

Before you install NetBlock, make sure your computer meets these requirements:

- Operating System: Windows 10 or later (64-bit recommended)
- RAM: At least 2 GB
- Disk Space: About 50 MB free
- Administrator access: Needed to install and run the app
- Internet connection: Required only for downloading and initial setup

NetBlock works on both laptops and desktops running Windows. It does not support Mac or Linux.

## 🔧 How to Use NetBlock

NetBlock is designed for ease of use. Here’s how to block internet access for apps:

1. **Open NetBlock** if it is not already running.
2. You will see a list of apps currently installed on your PC.
3. Find the app you want to block.
4. Click the toggle or checkbox next to that app to block its internet access.
5. The app is now blocked from using your internet.

You can unblock an app anytime by switching the toggle back off.

## 💡 Troubleshooting Common Issues

- **NetBlock does not start**  
  Restart your computer. Make sure you installed as an administrator.

- **App still has internet after being blocked**  
  Double-check you toggled the right app. Sometimes restart of the app or PC helps.

- **Installer won’t run**  
  Confirm you have Windows 10 or later. Disable antivirus temporarily if blocked.

- **NetBlock uses a lot of CPU**  
  NetBlock is built to be light. If it runs high, restart your PC and check for updates.
